# Third-Party Notices

## Code Dependencies

- Rust crates declared in `sidecar/Cargo.toml`
- Microsoft C++ build toolchain and Windows SDK, redistributable under their own terms
- `pypinyin` (MIT), used by `tools/lexicon-builder` to derive phrase pinyin during lexicon generation

## Data and Model Policy

- Base Chinese lexicon strategy is split from source code licensing.
- `jieba` default dictionary (MIT) is currently used as the primary source for generated alpha lexicon frequency data.
- `CC-CEDICT` is now used for phrase pronunciation correction and phrase supplementation. It is licensed under CC BY-SA 4.0 and generated lexicon artifacts derived from it must retain appropriate attribution and share-alike handling.
- OpenCC may be used later for script conversion and variant normalization. OpenCC is Apache-2.0.
- Model files such as Qwen GGUF are not redistributed by default from this repository; download policy is handled by `build/Download-Model.ps1`.
